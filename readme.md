# 官方评分准则：

总分为 **20 分**，共分为 5 个维度，每个维度 **4 分**。

## 1. 问题理解与项目目标（4 分）

重点考察项目是否清楚说明了：
- 要解决的问题是什么；
- 项目背景是什么；
- 项目目标是什么；
- 项目范围是否明确。

高分要求：问题、背景、目标和范围清晰，体现出对项目的深入理解。

## 2. 最终成果与完成度（4 分）

重点考察项目最终是否形成了完整成果，包括：
- 是否完成了预期任务；
- 成果是否完整；
- 质量是否较高；
- 是否具有说服力。

高分要求：成果完整、质量高，达到或超过预期。

## 3. 方法论、技术质量与分析（4 分）

重点考察项目采用的方法是否合理，包括：
- 方法是否适合该问题；
- 技术实现是否扎实；
- 分析过程是否充分；
- 方法选择是否有解释和依据。

高分要求：方法合理、实现扎实、分析充分，并能说明为什么这样做。

## 4. 影响、洞察与实际价值（4 分）

重点考察项目是否能说明其价值，包括：
- 得到了什么有意义的发现；
- 这些发现说明了什么；
- 项目有什么实际应用价值；
- 对现实问题有什么帮助或启发。

高分要求：能清楚展示项目洞察、影响和实际意义。

## 5. 展示质量（4 分）

重点考察最终展示是否清楚专业，包括：
- 结构是否清晰；
- 表达是否流畅；
- 视觉呈现是否专业；
- 时间控制和团队配合是否良好。

高分要求：展示有逻辑、表达清楚、专业流畅。
